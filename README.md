# 🏫 FikClass - Student Management System

**ระบบจัดการข้อมูลนักเรียนสำหรับโรงเรียน (Thai Schools)**

## ✨ ฟีเจอร์หลัก

✅ **จัดการนักเรียน** - เพิ่ม แก้ไข ลบข้อมูลนักเรียน  
✅ **บันทึกการเข้าเรียน** - ติดตามสถานะ มา/ขาด/ลา  
✅ **คะแนนและออมเงิน** - บันทึกคะแนนเก็บและเงินออม  
✅ **แดชบอร์ดสรุป** - แสดงสถิติรวมทั้งชั้น  
✅ **Dark/Light Mode** - รองรับสองโหมดมืด/สว่าง  
✅ **ค้นหานักเรียน** - ค้นหารายชื่อได้ทันที  
✅ **Local Storage** - เก็บข้อมูลในเครื่องอัตโนมัติ  
✅ **Responsive Design** - ใช้ได้บนมือถือและคอมพิวเตอร์  

## 🚀 วิธีใช้งาน

### 1. Clone Repository
```bash
git clone https://github.com/whatthefik007/fikclass.git
cd fikclass
```

### 2. เปิดไฟล์
```bash
# เปิด index.html ด้วย Browser
open index.html
```

หรือ **ใช้ Live Server** (ถ้าใช้ VS Code):
- ติดตั้ง Live Server Extension
- คลิกขวา → Open with Live Server

## 📁 โครงสร้างไฟล์

```
fikclass/
├── index.html      # หน้า HTML หลัก
├── style.css       # ไฟล์ Style CSS
├── script.js       # ไฟล์ JavaScript
├── README.md       # ไฟล์นี้
└── images/         # โฟลเดอร์รูปภาพ (logo)
```

## 🎨 ฟีเจอร์ที่อัปเดต

### ✅ ปัญหาที่แก้ไข (Bug Fixes)
- ✓ เพิ่ม Validation เพื่อตรวจสอบเลขที่ซ้ำ
- ✓ ซ่อนฟอร์มหลังเพิ่มข้อมูลแล้ว
- ✓ เพิ่มข้อความแสดงเตือนที่ชัดเจน
- ✓ บันทึก Dark Mode ให้ยังคงอยู่หลังรีเฟรช
- ✓ ข้อมูลว่างเปล่าจะแสดง "ไม่มีข้อมูลนักเรียน"

### 🎯 ปรับปรุงการใช้งาน
- ✓ ปุ่มบันทึก/ยกเลิกในฟอร์มเพิ่มนักเรียน
- ✓ Toast Alert แสดงสถานะทั้งสำเร็จและข้อผิดพลาด
- ✓ ล้างฟอร์มอัตโนมัติเมื่อซ่อน

## 🛠️ เทคโนโลยีที่ใช้

- **HTML5** - โครงสร้าง
- **CSS3** - ตกแต่งเอฟเฟกต์ (Gradient, Backdrop-filter)
- **JavaScript** - ฟังก์ชันเบื้องหลัง
- **Kanit Font** - ฟอนต์ไทย
- **LocalStorage API** - เก็บข้อมูล

## 📝 วิธีการใช้งานพื้นฐาน

### เพิ่มนักเรียนใหม่
1. คลิกปุ่ม "➕ เพิ่มนักเรียน"
2. กรอกข้อมูลทั้ง 6 ช่อง
3. คลิก "💾 บันทึก"

### แก้ไขข้อมูล
1. คลิกปุ่ม "📝 แก้ไข" ในแถวนักเรียน
2. แก้ไขข้อมูลใน Modal
3. คลิก "บันทึก"

### ลบข้อมูล
1. คลิกปุ่ม "🗑️ ลบ" ในแถวนักเรียน
2. ยืนยันการลบ
3. ข้อมูลจะถูกลบออก

### เปลี่ยนสถานะการเข้าเรียน
1. คลิกปุ่ม มา/ขาด/ลา ในคอลัมน์สถานะ
2. สถานะจะเปลี่ยนทันที

## 🔧 ตั้งค่าและอื่น ๆ

- **Dark Mode** - คลิกปุ่มด้านขวาบนเพื่อสลับโหมด
- **ค้นหา** - พิมพ์ชื่อในช่องค้นหาเพื่อกรองรายชื่อ
- **Dashboard** - ดูสถิติรวม (นักเรียนทั้งหมด มาเรียนวันนี้ คะแนนเฉลี่ย ยอดออมรวม)
